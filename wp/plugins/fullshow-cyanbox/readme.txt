A minimal Gutenberg block that displays a cyan-bordered box.

== Description ==
Adds a single dynamic block that renders a cyan-bordered box. No build tools required.

== Installation ==
1. Place the `fullshow-cyanbox` folder in `/wp-content/plugins/`.
2. Activate the plugin via the 'Plugins' screen in WordPress.
3. Add the **Fullshow Cyan Box** block to any post or page.
